- Public and Provate IP range:
PUBLIC RANGE
    - Class A: 0 - 127 ; 127 is reserved for loopback ; loopback is used to check if the network card is working or not by pinging
    - Class B: 128 - 191
    - Class C: 192 - 223
    - Class D: 224 - 239
    - Class E: 240 - 255

    Total range = 255
PRIVATE RANGE
    - Class A: 10.0.0.0 - 10.255.255.255
    - Class B: 172.16.0.0 - 172.31.255.255
    - Class C: 192.168.0.0 - 192.168.255.255
